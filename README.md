## Features
* Displaying a list of Pins
* Adding Pin
* Generating PIN
* Deleting Pin
* Changing Pin Visibility
* Retaining data in the database
* Dark mode
* Dynamic color (Android 12+)

## Stack
* Kotlin
* Coroutines
* Jetpack Compose
* Hilt
* KSP
* Room
* Junit5
* Mockk
* Turbine

## Technical Stuff
* Gradle Convention Plugins
* Multimodule app
* Android-free Domain (clean Kotlin JVM Library)
* Dependency Injection
* Unit Tested
* Detailed UX
